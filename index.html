Game Jams

These are the results of me participating in Game Jams alone or in a team of two. Check them out on my itch.io page!

Project Aleph

My first ever game jam submission for the Acerola Jam 0 with the theme "Aberration". Project Aleph is a 3D puzzle game inspired by the mobile game "That Level Again", where each level slight changes in the environment drasticaly change the puzzle solution. I developed this game solo in Godot in 3 weeks.

The Blobman [on Fire] in the Miniature Dungeon

This game was made in a team of two for the GMTK Game Jam 2024 with the theme "Built to Scale". I did both programming and music and after 4 days, and again working with Godot, I finished my second Game Jam Project.

Jeremy from HR

Jeremy from HR is a 2D clicker advaneture (inspired by old flash games I used to play) created for the Brackeys Game Jam 2025.1 within one week. Creating this game was again a team effort with me doing programming and music with one other member being responsible for art, sfx and writing. The theme of the jam was "Out of control", so we tried to realize this by putting the player in a situation that spirals 'out of control' over time, which is also supported by the story.

Automated assignment grading

Together with a fellow student I am creating an automated grading system for a computer graphics course at my university. The assignments are done in javascript and glsl. The automation works by per-pixel comparisons of the canvases the submissions are rendered into, which is why reworking some exercises is part of this project as well to enable such comparisons.
